# Bilink

原生的哔哩哔哩消息自动回复工具

## 测试版！

目前在测试阶段，仅供参考，非项目最终结构

## 使用

克隆项目，运行_init_或在你的测试函数中直接import包bilink

在listening函数中，默认：向当前登录用户私信发送“`你好`”，则自动回复“
